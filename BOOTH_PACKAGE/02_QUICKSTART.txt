Avatar Audio Safety Guard
クイックスタート

最短手順:
1. アバタールートに AvatarAudioSafetySettings を追加します
2. 動作モードはまず「診断のみ」のままにします
3. 必要なら判定プロファイルや診断オプションを調整します
4. 「AudioSource を走査」を押します
5. 「検出された AudioSource」一覧と「診断レポートを開く」で結果を確認します
6. 問題がなければ、そのまま運用します
7. Build 時だけ補正したい場合だけ「Build時に補正」に切り替えます

使い分け:
- 診断のみ:
  Build 時も値を変更しません
- Build時に補正:
  Build 時だけ build clone 側へ補正します
- 報告のみ:
  問題は表示しますが補正しません

まず Dry Run:
- 最初は必ず Dry Run で結果を確認してください
- いきなり Build 補正を前提にせず、一覧とレポートで内容を見るのがおすすめです

補足:
- Custom Rolloff は自動補正しません
- このツールは非破壊で動作し、元アバターには直接変更を加えません
- 必要な補正は Build 時の一時的な build clone にのみ適用されます
- 詳しい制限事項は README を確認してください
