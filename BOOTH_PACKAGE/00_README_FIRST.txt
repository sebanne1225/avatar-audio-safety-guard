Avatar Audio Safety Guard
BOOTH 配布パッケージ案内
version: 1.0.1

このツールは、VRChat アバター内の AudioSource を診断し、
公開前に危険な設定や過大な値を見つけやすくする Unity / VRChat 向けツールです。

主な方針:
- まず Dry Run で診断します
- このツールは非破壊で動作し、元アバターには直接変更を加えません
- 必要な補正は Build 時の一時的な build clone にのみ適用されます

最初にやること:
1. repo の README を確認してください
2. まずは VCC / VPM で導入してください
3. 導入後は「診断のみ」のまま Dry Run を試してください

導入の基本:
- 主導線は VCC / VPM です
- Git URL 導入は補助扱いです
